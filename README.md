# 🐾 Pawdience

**Course project:** Front‑end web project  
**Description:** Static website built with HTML and CSS.

## 📌 Overview

Pawdience is a static website designed as a simple landing page showcasing pet information.  
The site emphasizes clean layout and easy navigation using only HTML and CSS.

## 🛠️ Technologies Used

- HTML
- CSS

## 📄 Description

The website includes:

- A welcoming homepage with pet‑related visual content  
- Navigation menu structure  
- Image slideshow gallery  
- Responsive layout suitable for informational use

## 🎯 Key Takeaways

- Front‑end development with HTML and CSS  
- Simple design and layout structure  
- Consistent styling and responsiveness
